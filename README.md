## My-blog博客系统API

### 项目介绍

博客系统API，是一个基于SpringBoot的博客系统，后端采用SpringMVC + SpringBoot + Mybatis-plus进行开发，使用 MD5等工具类实现登录验证，使用Shiro做权限认证，使用PageHelper工具做分页处理，使用Slf4j做日志处理。

### 技术选型

- JDK 17
- MYSQL   
- Spring-Boot   
- Mybatis-plus
- Shiro
- Maven
- Lombok
